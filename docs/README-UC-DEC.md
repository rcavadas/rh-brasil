# Sistema de RH para o Mercado Brasileiro

# Pacote UC-DEC - Decimo Terceiro

## Versao

1.0

## Objetivo

Este pacote consolida a documentacao do decimo terceiro salario, cobrindo avos, parcelas, medias, encargos, rescisao e integracao com eSocial.

---

# Arquivos

| Caso de Uso | Descricao |
|---|---|
| UC-DEC-001 - Apurar Avos de Decimo Terceiro | Apuracao de avos e elegibilidade. |
| UC-DEC-002 - Calcular Primeira Parcela | Cálculo da primeira parcela. |
| UC-DEC-003 - Calcular Segunda Parcela | Cálculo da segunda parcela. |
| UC-DEC-004 - Calcular Medias de Verbas Variaveis | Medias de variaveis remuneratorias. |
| UC-DEC-005 - Calcular Encargos de Decimo Terceiro | INSS, FGTS e impactos correlatos. |
| UC-DEC-006 - Antecipar Decimo Terceiro nas Ferias | Adiantamento nas ferias. |
| UC-DEC-007 - Calcular Decimo Terceiro Rescisorio | Reflexos do desligamento. |
| UC-DEC-008 - Gerar Demonstrativo de Decimo Terceiro | Demonstrativo e auditoria. |
| UC-DEC-009 - Fechar Folha de Decimo Terceiro | Fechamento da competencia de 13o. |
| UC-DEC-010 - Integrar Decimo Terceiro ao eSocial | Eventos e transmissao governamental. |

---

# Escopo Funcional

- Avos e elegibilidade
- Primeira e segunda parcela
- Medias variaveis
- Encargos
- Adiantamento em ferias
- Rescisorio
- Demonstrativo
- Fechamento e eSocial
