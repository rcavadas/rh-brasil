# Sistema de RH para o Mercado Brasileiro

# Regras de Negócio (RN)

## Bloco 11 – Portal do Gestor

### Versão

1.0

### Objetivo

Este documento descreve as regras de negócio relacionadas ao portal do gestor, contemplando gestão da equipe, aprovações operacionais, indicadores gerenciais, acompanhamento de movimentações e dashboards de RH.

---

# Acesso e Permissões

## RN-1351

O gestor deve possuir acesso individual ao portal.

## RN-1352

O acesso deve respeitar a hierarquia organizacional vigente.

## RN-1353

O gestor deve visualizar apenas colaboradores sob sua responsabilidade.

## RN-1354

O sistema deve permitir múltiplos níveis de gestão.

## RN-1355

O sistema deve permitir delegação temporária de responsabilidades.

## RN-1356

A delegação deve possuir vigência definida.

## RN-1357

O sistema deve registrar histórico das delegações.

## RN-1358

O sistema deve controlar permissões por perfil.

## RN-1359

Toda concessão de acesso deve ser auditável.

## RN-1360

Todo acesso deve possuir trilha de auditoria.

---

# Gestão da Equipe

## RN-1361

O gestor deve visualizar a estrutura da sua equipe.

## RN-1362

O gestor deve visualizar dados básicos dos colaboradores autorizados.

## RN-1363

O gestor deve visualizar cargo e função dos colaboradores.

## RN-1364

O gestor deve visualizar lotação organizacional.

## RN-1365

O gestor deve visualizar situação contratual.

## RN-1366

O gestor deve visualizar admissões recentes.

## RN-1367

O gestor deve visualizar desligamentos recentes.

## RN-1368

O sistema deve permitir filtros por equipe.

## RN-1369

O sistema deve permitir pesquisa de colaboradores.

## RN-1370

Toda consulta deve ser auditável.

## RN-1378

## RN-1379

O sistema deve notificar o colaborador sobre a decisão.

## RN-1380

Toda movimentação deve ser auditável.

## RN-1422

## RN-1423

## RN-1424
