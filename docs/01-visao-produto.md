# Visão do Produto

O RH Brasil é uma plataforma SaaS de gestão do ciclo de vida do colaborador.

## Objetivos
- Centralizar informações de RH.
- Garantir conformidade legal.
- Automatizar processos operacionais.
- Reduzir retrabalho.

## Módulos previstos
- Recrutamento
- Admissão
- Cadastro
- Documentos
- Ponto
- Benefícios
- Férias
- Desenvolvimento
- Desligamento
