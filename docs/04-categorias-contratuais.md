# Categorias Contratuais

## CLT

Campos obrigatórios:
- Cargo
- Salário
- Jornada
- Data de admissão

Regras:
- Controle de férias
- Controle de ponto
- eSocial obrigatório

## Aprendiz

Campos:
- Entidade formadora
- Curso
- Data início
- Data término

Regras:
- Alerta de vencimento
- Controle de cotas
- Contrato específico

## Estagiário

Campos:
- Instituição de ensino
- Supervisor
- Seguro
- Termo de compromisso

Regras:
- Controle de recesso
- Vigência obrigatória

## PCD

Campos:
- Indicador PCD
- Necessidades de adaptação

Regras:
- Controle de cotas
- Proteção LGPD para documentação sensível

## Temporário

Campos:
- Empresa contratante
- Data final prevista

Regras:
- Controle de prazo contratual

## Terceirizado

Campos:
- Empresa prestadora
- Gestor responsável

Regras:
- Separação de vínculo da contratante
