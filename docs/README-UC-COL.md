# Sistema de RH para o Mercado Brasileiro

# Pacote UC-COL - Portal do Colaborador

## Versao

1.0

## Objetivo

Este pacote consolida a documentacao do portal do colaborador, cobrindo autosservico, consulta, solicitacoes e assinatura eletrônica.

---

# Arquivos

| Caso de Uso | Descricao |
|---|---|
| UC-COL-001 - Acessar Portal do Colaborador | Entrada e autenticacao no portal. |
| UC-COL-002 - Consultar Dados Cadastrais | Consulta de dados e historico. |
| UC-COL-003 - Solicitar Atualizacao Cadastral | Fluxo de alteracao cadastral. |
| UC-COL-004 - Consultar Holerite | Consulta de holerites e historico. |
| UC-COL-005 - Consultar Informe de Rendimentos | Consulta do informe anual. |
| UC-COL-006 - Solicitar Ferias | Pedido de ferias pelo colaborador. |
| UC-COL-007 - Consultar Banco de Horas | Consulta do saldo e movimentacoes. |
| UC-COL-008 - Consultar Beneficios | Visualizacao de beneficios ativos. |
| UC-COL-009 - Abrir Solicitacao ao RH | Chamados e solicitações internas. |
| UC-COL-010 - Assinar Documento Eletronico | Aceite e assinatura de documentos. |

---

# Escopo Funcional

- Autosservico
- Dados cadastrais
- Holerite e informe
- Ferias
- Banco de horas
- Beneficios
- Solicitações ao RH
- Assinatura eletrônica
