# Sistema de RH para o Mercado Brasileiro

# Pacote UC-PER - Avaliacao de Desempenho

## Versao

1.0

## Objetivo

Este pacote consolida a documentacao de avaliacao de desempenho, cobrindo ciclos, modelos, competencias, metas, feedback, PDI e calibracao.

---

# Arquivos

| Caso de Uso | Descricao |
|---|---|
| UC-PER-001 - Criar Ciclo de Avaliacao | Inicio do ciclo avaliativo. |
| UC-PER-002 - Executar Avaliacao 90 | Avaliacao direta. |
| UC-PER-003 - Executar Avaliacao 180 | Avaliacao com maior abrangencia. |
| UC-PER-004 - Executar Avaliacao 360 | Avaliacao completa multivisao. |
| UC-PER-005 - Avaliar Competencias | Competencias e comportamentos. |
| UC-PER-006 - Avaliar Metas | Metas e resultados. |
| UC-PER-007 - Registrar Feedback Continuo | Feedback ao longo do ciclo. |
| UC-PER-008 - Criar PDI | Plano de desenvolvimento individual. |
| UC-PER-009 - Calibrar Resultados | Calibracao entre avaliadores. |
| UC-PER-010 - Gerar Matriz Desempenho-Potencial | Mapa 9-box ou equivalente. |

---

# Escopo Funcional

- Ciclos de avaliacao
- Metas e competencias
- Feedback
- PDI
- Calibracao
- Matriz desempenho-potencial
