# Sistema de RH para o Mercado Brasileiro

# Regras de Negócio (RN)

## Bloco 17 – Avaliação de Desempenho

### Versão

1.0

### Objetivo

Este documento descreve as regras de negócio relacionadas à gestão de desempenho, avaliações periódicas, competências, metas, feedback contínuo, planos de desenvolvimento individual (PDI), calibração de resultados, sucessão e indicadores de performance.

---

# Ciclos de Avaliação

## RN-2001

O sistema deve permitir criação de ciclos de avaliação.

## RN-2002

Cada ciclo deve possuir período definido.

## RN-2003

O sistema deve permitir ciclos anuais.

## RN-2004

O sistema deve permitir ciclos semestrais.

## RN-2005

O sistema deve permitir ciclos trimestrais.

## RN-2006

O sistema deve permitir ciclos personalizados.

## RN-2007

O sistema deve registrar data de abertura.

## RN-2008

O sistema deve registrar data de encerramento.

## RN-2009

O sistema deve controlar status do ciclo.

## RN-2010

Toda movimentação deve ser auditável.
