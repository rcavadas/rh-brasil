# Sistema de RH para o Mercado Brasileiro

# Regras de Negócio (RN)

## Bloco 15 – Onboarding e Pré-Admissão

### Versão

1.0

### Objetivo

Este documento descreve as regras de negócio relacionadas ao processo de pré-admissão digital, onboarding corporativo, integração de novos colaboradores, provisionamento de recursos, acompanhamento do período de experiência e auditoria do processo de entrada.

---

# Pré-Admissão Digital

## RN-1801

O sistema deve permitir criação de processo de pré-admissão.

## RN-1802

O processo deve ser iniciado após aprovação do candidato.

## RN-1803

O sistema deve permitir envio de convite ao candidato.

## RN-1804

O convite deve possuir validade configurável.

## RN-1805

O sistema deve permitir reenvio do convite.

## RN-1806

O sistema deve registrar data de envio.

## RN-1807

O sistema deve registrar data de aceite.

## RN-1808

O sistema deve registrar status da pré-admissão.

## RN-1809

O sistema deve permitir cancelamento da pré-admissão.

## RN-1810

Toda movimentação deve ser auditável.

---

## RN-1811

## RN-1814

## RN-1817

## RN-1818

## RN-1819

## RN-1820

Toda movimentação deve ser auditável.

# Checklist Admissional

## RN-1831

O sistema deve possuir checklist admissional configurável.

## RN-1832

O checklist deve permitir tarefas obrigatórias.

## RN-1833

O checklist deve permitir tarefas opcionais.

## RN-1834

Cada tarefa deve possuir responsável.

## RN-1835

Cada tarefa deve possuir prazo.

## RN-1836

O sistema deve registrar conclusão das tarefas.

## RN-1837

O sistema deve registrar pendências.

## RN-1838

O sistema deve bloquear conclusão da admissão quando houver pendências críticas.

## RN-1839

O sistema deve permitir acompanhamento do checklist.

## RN-1840

Toda movimentação deve ser auditável.
