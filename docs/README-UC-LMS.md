# Sistema de RH para o Mercado Brasileiro

# Pacote UC-LMS - Treinamentos

## Versao

1.0

## Objetivo

Este pacote consolida a documentacao de treinamento e desenvolvimento, cobrindo cursos, trilhas, matriculas, certificados e reciclagem obrigatoria.

---

# Arquivos

| Caso de Uso | Descricao |
|---|---|
| UC-LMS-001 - Cadastrar Curso | Cadastro de cursos e conteudos. |
| UC-LMS-002 - Criar Trilha de Aprendizagem | Trilhas por cargo, competencia ou publico. |
| UC-LMS-003 - Matricular Colaborador | Matrícula em curso ou trilha. |
| UC-LMS-004 - Executar Treinamento | Execucao e registro de participacao. |
| UC-LMS-005 - Aplicar Avaliacao | Avaliacao do aprendizado. |
| UC-LMS-006 - Emitir Certificado | Emissao de certificado. |
| UC-LMS-007 - Controlar Reciclagem Obrigatoria | Reciclagem e vencimentos. |
| UC-LMS-008 - Vincular Curso a Competencia | Relacao com matriz de competencia. |
| UC-LMS-009 - Consultar Historico de Treinamento | Historico individual e gerencial. |
| UC-LMS-010 - Gerar Indicadores de Aprendizagem | Indicadores e analytics. |

---

# Escopo Funcional

- Cursos e trilhas
- Matriculas
- Execucao e avaliacao
- Certificacao
- Reciclagem obrigatoria
- Historico e indicadores
