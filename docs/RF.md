# Sistema de RH

## Especificação de Requisitos Funcionais (RF)

### Versão

1.0

### Objetivo

Este documento descreve os requisitos funcionais de um Sistema de Gestão de Recursos Humanos (RH) para atendimento às necessidades operacionais, trabalhistas, previdenciárias e fiscais do mercado brasileiro.

---

# RF – Requisitos Funcionais

## Módulo 1 – Cadastro de Colaboradores

### RF-001

O sistema deve permitir cadastrar colaboradores.

### RF-002

O sistema deve permitir editar dados cadastrais dos colaboradores.

### RF-003

O sistema deve permitir inativar colaboradores desligados.

### RF-004

O sistema deve permitir consultar histórico cadastral.

### RF-005

O sistema deve armazenar documentos digitais do colaborador.

### RF-006

O sistema deve permitir anexar múltiplos documentos por colaborador.

### RF-007

O sistema deve controlar versões dos documentos anexados.

### RF-008

O sistema deve registrar dependentes do colaborador.

### RF-009

O sistema deve controlar beneficiários para plano de saúde.

### RF-010

O sistema deve validar CPF junto à Receita Federal.

## Observação

Os requisitos funcionais descrevem exclusivamente as funcionalidades do sistema.

As regras legais, trabalhistas e previdenciárias devem ser descritas separadamente através das Regras de Negócio (RN), enquanto os aspectos técnicos e de qualidade devem ser tratados nos Requisitos Não Funcionais (RNF).
