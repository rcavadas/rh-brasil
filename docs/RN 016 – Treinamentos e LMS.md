# Sistema de RH para o Mercado Brasileiro

# Regras de Negócio (RN)

## Bloco 16 – Treinamentos e LMS

### Versão

1.0

### Objetivo

Este documento descreve as regras de negócio relacionadas à gestão de treinamentos, aprendizagem corporativa (LMS), trilhas de capacitação, certificações, treinamentos obrigatórios, integração com SST, avaliação de aprendizagem e indicadores de desenvolvimento.

---

# Catálogo de Cursos

## RN-1901

O sistema deve possuir catálogo centralizado de cursos.

## RN-1902

O sistema deve permitir cursos presenciais.

## RN-1903

O sistema deve permitir cursos online.

## RN-1904

O sistema deve permitir cursos híbridos.

## RN-1905

Cada curso deve possuir identificador único.

## RN-1906

Cada curso deve possuir descrição.

## RN-1907

Cada curso deve possuir carga horária.

## RN-1908

Cada curso deve possuir instrutor responsável quando aplicável.

## RN-1909

O sistema deve permitir categorização dos cursos.

## RN-1910

Toda movimentação deve ser auditável.

---

# Trilhas de Aprendizagem

## RN-1911

O sistema deve permitir criação de trilhas de aprendizagem.

## RN-1912

As trilhas devem agrupar múltiplos cursos.

## RN-1913

As trilhas devem possuir ordem configurável.

## RN-1914

O sistema deve permitir pré-requisitos.

## RN-1915

O sistema deve permitir trilhas obrigatórias.

## RN-1916

O sistema deve permitir trilhas opcionais.

## RN-1917

O sistema deve registrar progresso do colaborador.

## RN-1918

O sistema deve registrar conclusão da trilha.

## RN-1919

O sistema deve manter histórico das trilhas concluídas.

## RN-1920

Toda movimentação deve ser auditável.
