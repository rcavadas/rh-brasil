# Sistema de RH para o Mercado Brasileiro

# Regras de Negócio (RN)

## Bloco 10 – Portal do Colaborador

### Versão

1.0

### Objetivo

Este documento descreve as regras de negócio relacionadas ao portal de autosserviço do colaborador, contemplando autenticação, perfil, documentos, solicitações, benefícios, férias, banco de horas e comunicação com o RH.

---

# Acesso e Autenticação

## RN-1251

O colaborador deve possuir acesso individual ao portal.

## RN-1252

O login deve utilizar identificador único.

## RN-1253

O sistema deve permitir autenticação por e-mail corporativo.

## RN-1254

O sistema deve permitir autenticação por CPF quando configurado.

## RN-1255

O sistema deve exigir senha de acesso.

## RN-1256

O sistema deve permitir redefinição de senha.

## RN-1257

O sistema deve exigir política mínima de senha.

## RN-1258

O sistema deve permitir autenticação multifator (MFA).

## RN-1259

O sistema deve registrar tentativas de acesso.

## RN-1260

Todo acesso deve possuir trilha de auditoria.

---

# Perfil do Colaborador

## RN-1261

O colaborador deve poder consultar seus dados cadastrais.

## RN-1262

O sistema deve exibir histórico de dados cadastrais quando permitido.

## RN-1263

O colaborador deve poder solicitar atualização cadastral.

## RN-1264

Atualizações devem seguir fluxo de aprovação quando configurado.

## RN-1265

O sistema deve validar dados obrigatórios.

## RN-1266

O sistema deve registrar histórico das alterações.

## RN-1267

Toda alteração deve possuir data e hora.

## RN-1268

O sistema deve registrar responsável pela alteração.

## RN-1269

Dados sensíveis devem respeitar regras de acesso.

## RN-1270

Toda movimentação deve ser auditável.

---
