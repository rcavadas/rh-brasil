# Sistema de RH para o Mercado Brasileiro

# Pacote UC-ONB - Onboarding

## Versao

1.0

## Objetivo

Este pacote consolida a documentacao do onboarding, cobrindo pre-admissao, coleta documental, assinatura, acessos e experiencia inicial.

---

# Arquivos

| Caso de Uso | Descricao |
|---|---|
| UC-ONB-001 - Criar Processo de Pre-Admissao | Iniciar onboarding. |
| UC-ONB-002 - Enviar Convite ao Candidato | Convite e orientacao inicial. |
| UC-ONB-003 - Coletar Dados Cadastrais | Coleta de dados pessoais e bancarios. |
| UC-ONB-004 - Coletar Documentos Admissionais | Upload e validacao documental. |
| UC-ONB-005 - Executar Checklist Admissional | Checklist de pendencias. |
| UC-ONB-006 - Assinar Contrato de Trabalho | Assinatura do contrato digital. |
| UC-ONB-007 - Solicitar Provisionamento de Acessos | Acessos corporativos. |
| UC-ONB-008 - Solicitar Equipamentos | Equipamentos e recursos. |
| UC-ONB-009 - Atribuir Treinamentos Iniciais | Treinamentos de entrada. |
| UC-ONB-010 - Acompanhar Periodo de Experiencia | Acompanhamento do periodo inicial. |

---

# Escopo Funcional

- Pre-admissao
- Coleta documental
- Assinatura contratual
- Acessos e equipamentos
- Treinamentos iniciais
- Periodo de experiencia

## Estado no runtime

- A origem ATS ja converte candidato em pre-admissao rastreavel.
- A admissao agora registra dossie documental proprio, com documentos de onboarding e assinatura auditavel.
- O proximo refinamento natural do onboarding segue para provisionamento de acessos, equipamentos e trilha de experiencia inicial.
