# Sistema de RH para o Mercado Brasileiro

# Regras de Negócio (RN)

## Bloco 18 – Cargos, Salários e Carreira

### Versão

1.0

### Objetivo

Este documento descreve as regras de negócio relacionadas à estrutura organizacional, cargos, funções, tabelas salariais, progressões de carreira, promoções, sucessão, remuneração estratégica e equidade salarial.

---

# Estrutura Organizacional

## RN-2101

O sistema deve permitir cadastro da estrutura organizacional.

## RN-2102

A estrutura deve suportar múltiplos níveis hierárquicos.

## RN-2103

O sistema deve permitir cadastro de diretorias.

## RN-2104

O sistema deve permitir cadastro de gerências.

## RN-2105

O sistema deve permitir cadastro de coordenações.

## RN-2106

O sistema deve permitir cadastro de departamentos.

## RN-2107

O sistema deve permitir cadastro de equipes.

## RN-2108

Toda unidade organizacional deve possuir vigência.

## RN-2109

O sistema deve manter histórico estrutural.

## RN-2110

Toda alteração deve ser auditável.

---

# Cargos

## RN-2111

O sistema deve permitir cadastro de cargos.

## RN-2112

Todo cargo deve possuir identificador único.

## RN-2113

Todo cargo deve possuir descrição formal.

## RN-2114

Todo cargo deve possuir atribuições.

## RN-2115

Todo cargo deve possuir requisitos mínimos.

## RN-2116

Todo cargo deve possuir competências associadas.

## RN-2117

O sistema deve permitir controle de vigência dos cargos.

## RN-2118

O sistema deve manter histórico dos cargos.

## RN-2119

O sistema deve impedir exclusão de cargos utilizados.

## RN-2120

Toda alteração deve ser auditável.
