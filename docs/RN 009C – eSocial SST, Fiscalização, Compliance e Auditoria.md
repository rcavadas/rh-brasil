# Sistema de RH para o Mercado Brasileiro

# Regras de Negócio (RN)

## Bloco 09C – eSocial: SST, Fiscalização, Compliance e Auditoria Regulatória

### Versão

1.0

### Objetivo

Este documento descreve as regras de negócio relacionadas aos eventos SST do eSocial, monitoramento de conformidade, gestão de evidências, fiscalizações trabalhistas, compliance previdenciário e auditoria regulatória.

---

# Evento S-2210 – Comunicação de Acidente de Trabalho

## RN-1131

O sistema deve gerar evento S-2210 quando houver registro de CAT obrigatória.

## RN-1132

O evento deve utilizar informações registradas na CAT.

## RN-1133

O sistema deve validar os dados obrigatórios antes da transmissão.

## RN-1134

O sistema deve validar a identificação do trabalhador.

## RN-1135

O sistema deve validar data e hora do acidente.

## RN-1136

O sistema deve validar tipo do acidente.

## RN-1137

O sistema deve armazenar XML transmitido.

## RN-1138

O sistema deve armazenar XML de retorno.

## RN-1139

O sistema deve registrar recibo retornado.

## RN-1140

Toda transmissão deve possuir trilha de auditoria.
