# Sistema de RH para o Mercado Brasileiro

# Pacote UC-CAR - Cargos, Salarios e Carreira

## Versao

1.0

## Objetivo

Este pacote consolida a documentacao de cargos, salarios e carreira, cobrindo estrutura organizacional, faixas salariais, planos de carreira, equidade e sucessao.

---

# Arquivos

| Caso de Uso | Descricao |
|---|---|
| UC-CAR-001 - Cadastrar Estrutura Organizacional | Estrutura, niveis e relacoes. |
| UC-CAR-002 - Cadastrar Cargo | Cadastro do cargo. |
| UC-CAR-003 - Cadastrar Funcao | Cadastro da funcao. |
| UC-CAR-004 - Cadastrar Faixa Salarial | Faixas e limites. |
| UC-CAR-005 - Cadastrar Tabela Salarial | Tabela por nivel ou faixa. |
| UC-CAR-006 - Registrar Promocao | Evolucao vertical. |
| UC-CAR-007 - Registrar Progressao | Evolucao horizontal. |
| UC-CAR-008 - Criar Plano de Carreira | Plano de carreira e trilhas. |
| UC-CAR-009 - Monitorar Equidade Salarial | Equidade e analise comparativa. |
| UC-CAR-010 - Gerenciar Sucessao | Trilhas de sucessao e reserva. |

---

# Escopo Funcional

- Estrutura organizacional
- Cargos e funcoes
- Faixas e tabelas salariais
- Promocao e progressao
- Plano de carreira
- Equidade salarial
- Sucessao
