# Sistema de RH para o Mercado Brasileiro

# Pacote UC-GED - Gestao Documental

## Versao

1.0

## Objetivo

Este pacote consolida a documentacao da gestao documental, cobrindo tipos documentais, anexos, geracao, versionamento, assinatura e retencao.

---

# Arquivos

| Caso de Uso | Descricao |
|---|---|
| UC-GED-001 - Cadastrar Tipo Documental | Catalogo de tipos de documento. |
| UC-GED-002 - Anexar Documento ao Colaborador | Vinculo documental ao cadastro. |
| UC-GED-003 - Gerar Documento Automaticamente | Emissao automatica a partir de eventos. |
| UC-GED-004 - Versionar Documento | Controle de versao e historico. |
| UC-GED-005 - Assinar Documento Eletronicamente | Assinatura eletrônica simples. |
| UC-GED-006 - Assinar Documento com ICP-Brasil | Assinatura qualificada. |
| UC-GED-007 - Consultar Prontuario Eletronico | Consulta consolidada do dossiê. |
| UC-GED-008 - Aplicar Politica de Retencao | Retencao e expurgo. |
| UC-GED-009 - Descartar Documento Autorizado | Descarte controlado. |
| UC-GED-010 - Auditar Movimentacao Documental | Auditoria de acesso e alteracao. |

---

# Escopo Funcional

- Tipos documentais
- Anexos e prontuario
- Geracao automatica
- Versionamento
- Assinatura eletrônica
- Retencao e descarte
- Auditoria
