# Sistema de RH para o Mercado Brasileiro

# Pacote UC-GST - Portal do Gestor

## Versao

1.0

## Objetivo

Este pacote consolida a documentacao do portal do gestor, cobrindo visibilidade da equipe, aprovacoes e consultas gerenciais.

---

# Arquivos

| Caso de Uso | Descricao |
|---|---|
| UC-GST-001 - Visualizar Equipe | Visualizacao da equipe e estrutura. |
| UC-GST-002 - Aprovar Ferias | Aprovacao de ferias da equipe. |
| UC-GST-003 - Aprovar Ajuste de Ponto | Aprovacao de ajustes de ponto. |
| UC-GST-004 - Aprovar Horas Extras | Aprovacao de horas extras. |
| UC-GST-005 - Acompanhar Admissoes | Acompanhamento do ciclo admissional. |
| UC-GST-006 - Solicitar Desligamento | Solicitacao de desligamento. |
| UC-GST-007 - Consultar Indicadores da Equipe | Indicadores operacionais. |
| UC-GST-008 - Consultar Absenteismo | Absenteismo da equipe. |
| UC-GST-009 - Consultar Turnover | Rotatividade da equipe. |
| UC-GST-010 - Exportar Dados Autorizados da Equipe | Exportacao controlada de dados. |

---

# Escopo Funcional

- Visibilidade da equipe
- Aprovacoes
- Acompanhamento de admissoes e desligamentos
- Indicadores operacionais
- Exportacao autorizada
