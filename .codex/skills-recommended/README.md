# Skills recomendadas para este projeto

Use Skills para capacidades reutilizáveis entre projetos. Use `profiles` para contexto local deste repositório.

## Recomendadas

### hr-product-analyst
Para analisar requisitos, módulos e fluxos de RH corporativo no Brasil.

### lgpd-security-reviewer
Para revisar dados pessoais, dados sensíveis, logs, retenção, auditoria, permissões e integrações.

### backend-architecture-reviewer
Para revisar APIs, banco, autenticação, autorização, jobs, integrações e multiempresa.

### frontend-ux-reviewer
Para revisar telas, fluxos, acessibilidade, estados de erro/loading/vazio e experiência de RH/gestor/colaborador.

### infra-devops-reviewer
Para revisar Docker, deploy, ambientes, secrets, logs, backup, observabilidade e rollback.

### living-documentation-maintainer
Para manter documentação viva, decisões técnicas, riscos e histórico de sessão.

### handoff-memory-manager
Para garantir encerramento de sessão com memória persistente e retomada com perda mínima de contexto.

### qa-regression-reviewer
Para mapear testes, riscos de regressão e casos críticos.

## Como aplicar

- `AGENTS.md` orquestra o projeto atual.
- `.codex/profiles/` define papéis locais deste projeto.
- `.codex/` guarda memória e handoff.
- `docs/` guarda documentação viva.
- Skills globais devem guardar modos de operar reutilizáveis, não detalhes específicos deste repositório.
