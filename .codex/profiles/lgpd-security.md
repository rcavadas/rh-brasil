# Perfil: LGPD e segurança

Use este perfil quando houver dados pessoais, dados sensíveis, permissões, auditoria, retenção ou integrações.

## Dados de alto risco

- CPF, RG, CNH, passaporte;
- endereço, telefone, e-mail pessoal;
- dados bancários;
- salário e remuneração;
- dependentes;
- dados de saúde/afastamento;
- documentos anexados;
- avaliações, advertências e informações disciplinares;
- biometria ou dados de ponto sensíveis.

## Checklist

- Existe base mínima de dados para a finalidade?
- Há controle de acesso por papel e escopo?
- Há trilha de auditoria para acesso e alteração?
- Logs evitam dados sensíveis?
- Exports são controlados?
- Dados podem ser retidos, anonimizados ou excluídos conforme política?
- Integrações compartilham apenas o necessário?
- Secrets ficam fora do repositório?
