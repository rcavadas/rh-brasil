# Perfil: Infra e DevOps

Use este perfil para Docker, deploy, ambientes, CI/CD, observabilidade, backups e operação.

## Avaliar

- variáveis de ambiente;
- secrets;
- Dockerfile/Compose;
- healthchecks;
- logs;
- métricas;
- backup/restore;
- migrações em deploy;
- rollback;
- TLS/proxy;
- escalabilidade;
- filas/workers;
- storage de documentos;
- segregação de ambientes.

## Regras

- Não registrar secrets.
- Não fazer deploy sem confirmação.
- Não rodar migração destrutiva sem plano.
- Documentar comandos em `docs/INFRASTRUCTURE.md`.
