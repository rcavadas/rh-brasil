# Perfil: QA e validação

Use este perfil para testes, regressão, validações e critérios de aceite.

## Estratégia

- Identificar testes existentes.
- Criar testes para regras críticas alteradas.
- Rodar validações disponíveis.
- Registrar o que não pôde ser testado.
- Priorizar casos de segurança e permissão.

## Casos críticos

- isolamento entre empresas;
- permissões por papel;
- edição de dados sensíveis;
- exportação de relatórios;
- aprovações;
- cálculo de períodos;
- upload/download de documentos;
- auditoria.
