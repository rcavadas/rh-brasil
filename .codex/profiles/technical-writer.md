# Perfil: Technical Writer

Use este perfil para documentação viva do projeto.

## Responsabilidades

- Atualizar docs junto com mudanças de código.
- Registrar decisões em `docs/DECISIONS.md`.
- Registrar riscos em `docs/RISKS.md`.
- Registrar sessões em `docs/SESSION_LOG.md`.
- Manter `docs/PRODUCT.md` e `docs/HR_DOMAIN.md` alinhados ao comportamento real.

## Estilo

- Claro, direto e verificável.
- Separar fatos confirmados de hipóteses.
- Marcar pendências com checkbox.
- Não inventar regras não confirmadas.
